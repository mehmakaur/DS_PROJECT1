#  Restaurant Growth Advisor

A Data Science project focused on analyzing restaurant performance using customer ratings, cuisine preferences, and operational attributes. The project combines data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and Machine Learning to identify key factors driving restaurant success.

## Key Highlights

* Analyzed restaurant, customer, and cuisine datasets.
* Performed data cleaning, feature engineering, and visualization.
* Conducted SQL-based business intelligence analysis.
* Developed a Random Forest classifier to predict restaurant success categories.
* Identified key factors influencing customer satisfaction and restaurant performance.

## Tech Stack

**Python | SQL | Pandas | NumPy | Matplotlib | Seaborn | Scikit-Learn**

## Dataset

Restaurant Data with Consumer Ratings (Kaggle)

## Project Files

* `restaurant-project.ipynb` – Data Analysis & Machine Learning
* `restaurant_project.sql` – SQL Business Analysis Queries
